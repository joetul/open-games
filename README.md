# Open Games

A collection of free, open-source browser-based puzzle games inspired by the NY Times Games section. Built with vanilla HTML, CSS, and JavaScript.

## Games

- **Sudoku** - Classic 9x9 number puzzle with multiple difficulty levels

## Running Locally

Since this project uses ES modules, you need a local server:

```bash
python3 -m http.server
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

Alternatively, use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension.

## Deploying to GitHub Pages

1. Push this repo to GitHub (public repository)
2. Go to **Settings > Pages**
3. Set **Source** to "Deploy from a branch"
4. Select branch `main` and folder `/ (root)`
5. Your site will be live at `https://<username>.github.io/<repo-name>/`
